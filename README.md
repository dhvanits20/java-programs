# Java Programs Collection

This repository contains a collection of Java programs and solutions.

## Contents

The programs are organized by question number:
- **q1 to q36**: Various Java implementations covering basic to advanced concepts.
- **Resources**: [practicalsethjava.pdf](./practicalsethjava.pdf)

## How to use

1.  Navigate to any question folder (e.g., `q1`).
2.  Inside the `src` directory, you will find the `.java` source files.
3.  You can view the code directly here on GitHub.
